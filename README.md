# 프로젝트
### 요약
* 설명 : 2018년 국비지원 웹 개발자 양성과정에서 진행했던 세미 프로젝트(팀)
* 기간 : 약 한달(2018.04)
* 주제 : 개인별 맞춤 여행지 추천 및 공유 사이트
* 사용 기술 :
  - Java
  - HTML/CSS
  - JavaScript/Jquery/Ajax
  - Servlet/JSP
  - MVC Model 2
* 담당 부분 : 
  - 프로젝트 계획서 작성
  - 여행 일정 생성 관련 페이지 담당(DB 및 해당 코드 작성)
  - 시퀀스 다이어그램, 유스케이스 다이어그램 작성
